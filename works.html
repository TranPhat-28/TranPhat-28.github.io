Online file storage website

This is my lastest project with my classmate, where we built a functional web application and provided it with some security features.

  • Features: Login, signup, upload and download files, view your files and others' shared files, comment.
  • Security: Password hash, filter input (for SQL Injection and XSS), CSRF, file scan.

My contribution: Building the majority of frontend and backend (including many features of the web), building database, applying some security measurements.

Technologies used:

  • Frontend: HTML, CSS, Bootstrap, Javascript.
  • Backend: Ubuntu server, Apache, MySQL, PHP.
